What if a color changes in the design?

= LOTS of repetitive work!

Enter Font Awesome!

  • No JPGs, No PNGs, No GIFs.
  • No static (non-flexible) images of any kind!
  • Font Awesome is a “font” that gives you icons.
    • “Someone finally found a use for the WingDings font!”
      ...kind of. :)
  • Use CSS to alter typefaces in your HTML
    • (size, color, borders, etc)
  • Same idea, same concept.

How do I install Font Awesome?

  • Go to fontawesome.io
  • Click download.
  • Unzip that folder and stick it on your website.
<script defer src="/fontawesome/fontawesome-all.js"></script>
