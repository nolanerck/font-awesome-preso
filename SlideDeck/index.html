So what is FontAwesome?

How many icons in 1 webapp?

What if a color changes in the design?

Does the site have seasonal color changes?

= LOTS of repetitive work!

Enter Font Awesome!

  • No JPGs, No PNGs, No GIFs.
  • No static (non-flexible) images of any kind!
  • Font Awesome is a “font” that gives you icons.
    • “Someone finally found a use for the WingDings font!”
      ...kind of. :)
  • Use CSS to alter typefaces in your HTML
    • (size, color, borders, etc)
  • Same idea, same concept.
  • Free and Pro versions available

How do I install Font Awesome?

  • Go to fontawesome.com
  • Click download.
  • Unzip that folder and stick it on your website.
<script src="/fontawesome/fontawesome-all.js"></script>

How do I install Font Awesome?

Can also just use their CDN:

<script src="https://use.fontawesome.com/releases/v5.0.6/js/all.js"></script>

Using Font Awesome

Just use the <i> HTML tag wherever you want an icon:

<i class="fas fa-camera-retro"></i>

"fas" is the default style, "fa-camera-retro" is the icon itself

Using Font Awesome

Four styles available:
Solid (fas), Regular (far), Light (fal), Brands (fab)

Regular and Light are only in the Pro version

So What's the Catch?

  • 30MB download
  • Some extra overhead in your site.
    fontawesome-all.js is 700K by itself
  • Pro version is subscription based
  • Styling the I or parent DIV to customize
  • Rare CSS quirks
    had to use in-line CSS to customize some things

Performance?

  • Really good
  • Docs say up to 1000 icons per page with no noticable lags
  • After that, not sure
  • When would you ever really need more than 1000 icons per page?
  • Large scale tile based games?

Taking It Further

  • This is all just CSS
  • Can use it in SASS, LESS, jQuery, Vanilla JavaScript, etc
  • All the CSS knowledge you already have still works
  • All the JavaScript tricks you currently use still work too
  • Works in Bootstrap, Angular, Vue.js, etc
  • Remember: the browser just sees a font and CSS
