So what is FontAwesome?

So what is FontAwesome?

Building web apps = making icons
...LOTS of icons!

Icons in different sizes...

And colors...

Orientations...

And hover states...

With borders...

And shadows...

1 icon could = 10+ images.

How many icons in 1 webapp?

What if a color changes in the design?

= LOTS of repetitive work!

Enter Font Awesome!

  • No JPGs, No PNGs, No GIFs.
  • No static (non-flexible) images of any kind!
  • Font Awesome is a “font” that gives you icons.
    • “Someone finally found a use for the WingDings font!”
      ...kind of. :)
  • Use CSS to alter typefaces in your HTML
    • (size, color, borders, etc)
  • Same idea, same concept.
  • Free and Pro versions available

How do I install Font Awesome?

  • Go to fontawesome.com
  • Click download.
  • Unzip that folder and stick it on your website.
<script defer src="/fontawesome/fontawesome-all.js"></script>

How do I install Font Awesome?

Can also just use their CDN:

<script defer
src="https://use.fontawesome.com/releases/v5.0.6/js/all.js"></script>

Using Font Awesome

Just use the HTML tag wherever you want an icon: 

<i class="fas fa-camera-retro"></i>

"fas" is the default style, "fa-camera-retro" is the icon itself

Four styles available: Solid (fas), Regular (far), Light (fal), Brands (fab)

Regular and Light are only in the Pro version
